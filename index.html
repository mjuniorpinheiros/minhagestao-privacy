Política de Privacidade — Minha Gestão

Última atualização: junho de 2026

1. Dados coletados

O Minha Gestão coleta apenas os dados necessários para o funcionamento do app:

  • Email e nome (via login Google ou email/senha) para autenticação
  • Dados financeiros, tarefas, eventos e registros de tempo inseridos pelo usuário

2. Uso dos dados

Os dados são usados exclusivamente para fornecer as funcionalidades do app. Não vendemos nem compartilhamos seus dados com terceiros.

3. Armazenamento

Os dados ficam armazenados localmente no dispositivo e, quando sincronizados, no Firebase (Google Cloud) com criptografia.

4. Anúncios

A versão gratuita exibe anúncios via Google AdMob. O AdMob pode coletar dados de uso conforme sua própria política de privacidade.

5. Assinatura Premium

O pagamento da assinatura é processado pela Google Play. Não armazenamos dados de cartão de crédito.

6. Seus direitos

Você pode excluir sua conta e todos os dados a qualquer momento nas configurações do app.
